# HTMLファイルを書いてみよう

プロフィールカードの構造をHTMLで記述します。  
以下のHTMLを`index.html`として保存してください。

```html
<!DOCTYPE html>
<html lang="ja">

<head>
  <meta charset="UTF-8">
  <title>プロフィール</title>
</head>

<body>
  <div>
    <div>
      <img src="image.jpg" alt="プロフィール画像">
      <div>
        <h1>名前</h1>
      </div>
    </div>
    <div>
      <p>自己紹介文</p>
    </div>
  </div>
</body>

</html>
```
